Вы должны добавлять в объект каждое поле с задержкой в 1 секунду. Т.е. объект должен создаваться 4 секунды и только после этого, как объект будет готов, функция должна возвращать результат. (см. setTimeOut() и создание объекта) Добавьте в вашу функцию вызов value = confirm(auto); Соответственно, ваша функция должна возвращать объект только при согласии пользователя. Если пользователь не подтвердил, ваша функция должна возвращать строку “Ошибка” В JS функция может возвращать разные типы!! Написать функцию, которая возвращает промис с результатом выполнения функции написанной в п.1-2 Обработать (promise.then() и promise.catch()) созданного промиса и вывести в консоль результат (либо ошибку, либо объект)
